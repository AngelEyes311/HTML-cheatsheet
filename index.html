HTML Tags

Tag Name Description
<html> html document root tag The <html> tag is the root element displayed at the start and end of an HTML document.
<head> head Information about the web page (metadata), like the title, and links to the stylesheets and scripts.
<title> title Displays the title of the page.
<body> body the substance of the document, all visable content.
<h1>, <h2>, <h3>... heading Headings and sub-headings, <h1> to <h6> tags, used for SEO and sectioning of the topics
<p> paragraph the paragraph tag, text or bulk of information on page
<img> image Images (<img>) can be added by difining a src to go with it.
<a> anchor The <a> anchor element is used to create hyperlinks in an HTML document. The hyperlinks can point to other webpages, files on the same server, a location on the same page, or any other URL via the hyperlink reference attribute, href.
<div> division Short for “division” or a container that divides the page into sections.

HTML Global Attributes

Attribute Description
Accesskey Provides a hint for generating a keyboard shortcut for the current element.
Autocapitalize Controls whether and how text input is automatically capitalized as it is entered/edited by the user.
Class A space-separated list of the classes of the element.
contenteditable An enumerated attribute indicating if the element should be editable by the user. If so, the browser modifies its widget to allow editing.
contextmenu The id of a to use as the contextual menu for this element.
data Forms a class of attributes, called custom data attributes, that allow proprietary information to be exchanged between the HTML and its DOM representation that may be used by scripts
dir An enumerated attribute indicating the directionality of the element's text. It can have the following values: left to right, right to left auto.
draggable An enumerated attribute indicating whether the element can be dragged, using the Drag and Drop API.
dropzone An enumerated attribute indicating what types of content can be dropped on an element, using the Drag and Drop API.
hidden A Boolean attribute indicates that the element is not yet, or is no longer, relevant.
id Defines a unique identifier (ID) which must be unique in the whole document.
lang Helps define the language of an element: the language that non-editable elements are in, or the language that editable elements should be written in by the user.
